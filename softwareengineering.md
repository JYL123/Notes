# Software Engineering 
## Full stack
```
The capacity to understand every layer of a software product.
```
* System administration and networking: server, network, and hosting environment.
* Data Modeling
* Business Logic
* API layer / Action Layer / MVC
* User Interface
* User Experience
* Testing
